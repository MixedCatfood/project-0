About This Site:

Created by Carson Tucker. This site was made for players to have a complete compilation of frame data for Super Smash Bros. Melee. Currently, this website is not designed for mobile use; a mobile design may come in the future.

Most of the frame data was retreived from "Stratocaster's Hitbox System on Smashboards," and Mitchell Meier's "Melee Database".



Where do we go from here?

This project is ongoing and there are features that still need to be implemented, so expect updates in the following months!
